# These tools are used for local development and are NOT installed in

#

#   pip install -r requirements-dev.txt

#   ruff        — Linter + formatter (replaces black + isort + flake8)
#   bandit      — Python security linter (SAST)
#   mypy        — Static type checker
#   types-*     — Type stubs for mypy
#
# Note: pytest and test dependencies remain in per-service requirements.txt
# files because each service has different test dependencies (e.g., aiosqlite
# for DB-access services, postgres service containers for integration tests).
